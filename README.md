Version 2

This is the finished prototype, capable of taking user inputs and creating a manifest.json file, which then allows a minecraft mod to actually be recognised and work in game. 

The other files you see, os_test and json_structure do the following:

json_structure: 
Gives the main file the structure of the json file that needs to be written, with the main file controlling all the edits to the inputs

os_test:
This is a little file I used to play around with the OS module to see how it would fit into my program
